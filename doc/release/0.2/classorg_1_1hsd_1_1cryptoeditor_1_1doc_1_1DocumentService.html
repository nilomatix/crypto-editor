void org.hsd.cryptoeditor.doc.DocumentService.load ( File  file)
inline

Assembles a document from the passed file by reading the contents and decrypting them based on the wrapped encryption definition. This process uses LoadService and DecryptionService to perform expensive operations on seperate threads. Eventually the document is set as currently in editing for the application.
