Cryptographer org.hsd.cryptoeditor.crypto.CryptoService.getCryptographer ( Encryption  encryption,
char[]  password 
)
inline

Resolves keys or key-pairs by either generating or retrieving them from storage and assembles a cryptographer initialized with the passed encryption and the key material.

For Encryption with a PBE type: For encryptions where EncryptionType#isPBEType() is true, the key will be generated from the passed password using the method specified by the encryption type.

For Encryption with a symmetric type: For encryptions where EncryptionType#isAsymmetric() is false, a key will either be retrieved from local storage or generated.

For Encryption with an asymmetric type: For encryptions where EncryptionType#isAsymmetric() is true, the public get will be read from the passed encryption via Encryption#getPublicKey(). If the public key is present, the private key will be retrieved from local storage. Otherwise a new key-pair will be generated.

passwordeither for PBE key derivation or key retrieval from storage
