Detailed Description

Cryptographer implementation based on the Bouncy Castle Provider.

Note that you have to pass keys and key-pairs which work with the Bouncy Castle Provider (preferably created using it).

See also
org.bouncycastle.jce.provider.BouncyCastleProvider

void org.hsd.cryptoeditor.crypto.grapher.BCCryptographer.setEncryption ( Encryption  encryption)
inline

Sets the Encryption which will be used for cryptographic operations with this Cryptographer.

Notice that you also have to initialize the Cryptographer with a valid key via setKey(Key) (for symmetric encryption types) or key-pair via setKeyPair(KeyPair) (for asymmetric encryption types).
